## What is Howler.js?

*Howler.js* is an audio library for the modern web. It uses *Web Audio API* and falls back to *HTML5 Audio*.

## How you use howler.js?

```js
var sound = new Howl({
  src: ['sound.mp3']
});

sound.play();
```
