GPT3 Powered CLI
================

https://github.com/abhagsain/ai-cli

Get answers for CLI commands from GPT3 right from your terminal

## Installation

You'd need to install it globally

	$ npm i @abhagsain/ai-cli -g

Usage

	$ ai ask "What is SAML?"

## Commands

	$ ai model

	$ ai auth
